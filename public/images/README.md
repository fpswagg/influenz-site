# Images

Ce dossier contient les images utilisées sur le site.

## Structure

- `clients/` - Logos des clients (format recommandé: PNG transparent, 200x200px)
- `projects/` - Images des projets réalisés (format recommandé: JPG/PNG, 800x600px)

## Placeholders

Pour l'instant, les images sont remplacées par des placeholders visuels dans les composants.
Remplacez-les par vos propres assets une fois disponibles.
